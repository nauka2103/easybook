About EasyBooking

EasyBooking is a student project created to demonstrate modern web development using Express.js.

Our Goal

The goal of this project is to learn backend routing, form handling, and clean frontend structure using Express.js.

Technologies

HTML, CSS, JavaScript, Node.js, and Express.js were used to build this application.

Team Work

The project was developed by a small team where each member was responsible for a specific part of the system.
